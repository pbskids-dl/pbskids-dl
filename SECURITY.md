# Security Policy

## Supported Versions

Only the latest version and one version below the latest get security updates.

| Version | Supported          |
| ------- | ------------------ |
| 3.0     | :white_check_mark: |
| 3.1     | :white_check_mark: |
| > 3.0   | :x:                |
